#  CreaTuLanding1-Mansilla

#  Descripción
Primera entrega del curso de **React en Coderhouse**.  
El proyecto representa la base del e-commerce **Checkmate**, con los primeros componentes desarrollados en React.

---

#  Componentes incluidos
- **NavBar:** contiene el logo, los enlaces de navegación y el widget del carrito.
- **CartWidget:** ícono del carrito integrado en la barra de navegación.
- **ItemListContainer:** recibe un mensaje de bienvenida mediante **props**.

---

# Tecnologías utilizadas
- React (Vite)
- JSX
- CSS
